# 00_framework Directory Structure

## Top-Level Directories

```
src/
  components/
    window/          # Window system (spatial semantics)
      Window.tsx     # Main window component
      Window.css     # Spatial zone definitions
    desktop/         # Desktop container
      Desktop.tsx    # Desktop background
      Desktop.css    # Desktop styling
    launcher/        # Launcher UI
      Launcher.tsx
      Launcher.css
    clipboard/       # Clipboard UI
      ClipboardWindow.tsx
      ClipboardHandler.tsx
    feedback/        # Feedback window
      FeedbackWindow.tsx
  core/
    window/          # Window management
      windowManager.ts    # Window lifecycle
      windowTypes.ts      # Type definitions
      sessionMemory.ts    # Persistence
      windowDefaults.ts   # Defaults
    clipboard/       # Clipboard manager
      clipboardManager.ts
    ui/              # UI state management
      uiState.ts
      zoneRenderer.ts
      instrumentMap.ts
  styles/
    design-system.css  # Design tokens (root colors, etc.)
  system/
    SystemIgnition.tsx
  templates/
    BaseTemplate.tsx
    domain-templates/
```

## Key Files for Review

**Window System (Safe to Migrate):**
- `src/components/window/Window.tsx` - Spatial semantics
- `src/components/window/Window.css` - Zone definitions
- `src/core/window/windowManager.ts` - Lifecycle
- `src/core/window/windowTypes.ts` - Types
- `src/core/window/sessionMemory.ts` - Persistence

**Desktop System (Safe to Migrate):**
- `src/components/desktop/Desktop.tsx`
- `src/components/desktop/Desktop.css`

**Design System (Safe to Migrate):**
- `src/styles/design-system.css`

**UI Components (Review Before Migration):**
- `src/components/launcher/`
- `src/components/clipboard/`
- `src/components/feedback/`

**Core UI (Conditional - Review First):**
- `src/core/ui/` - Check for enforcement

**Templates (Conditional - Review First):**
- `src/templates/` - Check for policy logic
