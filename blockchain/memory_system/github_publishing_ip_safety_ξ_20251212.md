### 3. Decide IP Posture (Choose One or Combination)

**Option A: Defensive Publication**
- Publish enough to prevent others from patenting around you

- Prevents patent blocking

- File provisional (cheap, quiet)

- Keep core private
- Publish shells only
- System only works when all parts align
- Copying one part is useless
